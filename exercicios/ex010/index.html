Usando links externos

Link externo é um link que vai apontar para outra Site que não é nosso!

EX:

Você pode acesar o meu Perfil no intagram.

Quando se usa o link externo como o modelo acima no código o Site para qual o usuario sera encaminha se sobre poem ao seu site (ex: abre uma nova pagina encima da sua).
